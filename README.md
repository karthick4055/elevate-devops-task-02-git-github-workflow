# DevOps Internship – Task 2: Git & GitHub Fundamentals

## Objective
To understand and practice real-world version control workflow using Git and GitHub, including repository initialization, committing changes, pushing to remote, branching, merging, and rollback operations.

## Tools Used
- Git
- GitHub

## Tasks Completed
- Installed Git and configured global username and email
- Initialized a local Git repository and understood the purpose of `.git`
- Created files and tracked changes using:
  - `git status`
  - `git add`
  - `git commit`
- Created a GitHub repository and connected it with local repo (HTTPS)
- Pushed local commits to remote GitHub repository
- Created a feature branch, made changes, and merged it into the main branch
- Practiced rollback and history tracking using:
  - `git log`
  - `git reset`
  - `git checkout`
